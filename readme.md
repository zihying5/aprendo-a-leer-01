# Aprendo a Leer 📖

¿Qué es esto?

**Aprendo a Leer** es una pequeña herramienta de práctica de alfabetización en línea, diseñada para estudiantes principiantes de español — especialmente para quienes tienen poco tiempo de clase y tienden a olvidar rápido entre sesiones.

### La historia detrás del proyecto

Este proyecto nació de un momento real de frustración en un aula de voluntariado.

Trabajaba con estudiantes que tenían muy poca base de español, pocas horas de clase y muchas ganas de aprender — pero con la sensación constante de olvidar demasiado rápido.

A partir de mi experiencia docente y de mi investigación de máster, quise crear un recurso sencillo, accesible y con un ritmo muy lento, pensado para practicar también fuera del aula.

Tras muchas pruebas, ajustes y conversaciones con herramientas de IA, nació este pequeño prototipo.

No es un proyecto cerrado. Es un punto de partida.

Y quizás — algún día, aunque yo ya no esté — pueda servirle a alguien.

### Cómo usarlo

1. Entra al sitio: [https://zihying5.github.io/aprendo-a-leer-01/](https://zihying5.github.io/aprendo-a-leer-01/)
2. Elige una unidad (Unidad A: vocales / Unidad B: p y l)
3. Selecciona un nivel (Nivel 0 es repaso, los niveles 1–3 aumentan en dificultad)
4. Escucha el audio 👂 y escribe lo que oyes
5. Compara tu respuesta y pasa al siguiente ejercicio
6. ¡Gana estrellas ⭐ a medida que avanzas!
